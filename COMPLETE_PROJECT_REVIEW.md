## The Accelerate Method V3

### Framework: The Visibility Pyramid‚Ñ¢
A 6-phase progression model for getting promoted:

| Phase | Name | Core Concept |
|-------|------|-------------|
| 1 | **Competence Entry Fee** | Must deliver results in current role (baseline) |
| 2 | **Skip-Level Echo** | Manager's manager must know your name + impact |
| 3 | **The Bias Lever** | Reframe visibility from "self-promotion" to "value clarity" |
| 4 | **Genesis Protocol** | Launch signature initiative that showcases leadership |
| 5 | **Strategic Positioning** | Build relationships 6-12 months before promotion cycle |
| 6 | **The Trojan Horse** | Take on stretch responsibilities BEFORE promotion title |

### Key Concepts
- **Performance ‚â† Promotion:** Hard work alone doesn't get you promoted
- **Visibility is the missing variable:** You must be SEEN doing the work
- **Relationships matter:** 6-12 month lead time for promotion opportunities
- **The Trojan Horse Strategy:** Prove you can do the job by actually doing it without the title

## Discovery Call Protocol

- **Goal:** Diagnose ‚Üí Prescribe ‚Üí Commit (in 15 minutes)
- **Format:** 1-on-1 phone call
- **Outcome:** Determine if Launch/Pivot/Accelerate is right fit + enrollment decision

### Call Structure (15 min hard stop)
1. **Connection (1 min)** - Build rapport, acknowledge their frustration
2. **Diagnosis (4 min)** - Ask key questions about current situation
3. **Prescribe (5 min)** - Recommend path + explain why + show value
4. **Commit (5 min)** - Answer objections, move to enrollment

### Key Diagnostic Questions

- "What's been the biggest obstacle so far?"
- "What have you already tried?"
- "What would success look like to you?"
- "What would you do if coaching wasn't an option?"
- "How soon do you need to land/pivot/promote?"
