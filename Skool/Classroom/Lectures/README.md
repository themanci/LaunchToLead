# Skool Classroom Lectures

This folder contains outlines and resources for creating video lectures for the Launch to Lead Skool community.

**Source Document:** `/methods/The Launch Method V3.md`
**Total Methods:** 52 methods across 11 sections

## Recording Setup
- **Software:** OBS Studio
- **Format:** Screen share + face cam (picture-in-picture)
- **Target Length:** 5-10 minutes per method
- **Style:** Walkthrough of method materials with live explanation

## Production Workflow
1. Review method outline in `00-launch-method-content-outline.md`
2. Check section README for key points and estimated length
3. Open corresponding materials from `/methods/The Launch Method V3.md`
4. Record in OBS with screen share + face cam
5. Export and upload to Skool Classroom

## Naming Convention
Videos: `[Section#]-[Method#]-[MethodName].mp4`
Example: `01-01-job-sourcing-strategy.mp4`
