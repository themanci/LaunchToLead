# Skool Classroom Lectures

This folder contains outlines and resources for creating video lectures for the Launch to Lead Skool community.

## Recording Setup
- **Software:** OBS Studio
- **Format:** Screen share + face cam (picture-in-picture)
- **Target Length:** 5-10 minutes per method
- **Style:** Walkthrough of method materials with live explanation

## Folder Structure
```
Lectures/
├── 01-job-sourcing/
├── 02-networking/
├── 03-resume/
├── 04-cover-letter/
├── 05-interview-prep/
├── 06-remote-interview/
├── 07-career-fair/
├── 08-phone-screen/
├── 09-offer-negotiation/
├── 10-mental-resilience/
└── 11-special-cases/
```

## Production Workflow
1. Review method outline in this folder
2. Open corresponding HTML material from `/methods/` or `/client_materials/`
3. Record in OBS with screen share + face cam
4. Export and upload to Skool Classroom

## Naming Convention
Videos: `[Section#]-[Method#]-[MethodName].mp4`
Example: `01-01-job-sourcing-strategy.mp4`
