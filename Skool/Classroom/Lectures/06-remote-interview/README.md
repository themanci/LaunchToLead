### 6.1 The Digital Stage Presenceâ„¢
- **Length:** 5-7 min
- **Status:** Not recorded
- **Key Points:**
  - Technical setup checklist
    - Test camera, microphone, internet connection
    - Close unnecessary applications
    - Have backup plan (phone hotspot, dial-in number)
  - Visual environment
    - Clean, professional background
    - Good lighting (face the light source)
    - Camera at eye level
  - Audio optimization
    - Use headphones to reduce echo
    - Mute when not speaking in group settings
    - Quiet environment
  - Virtual presence
    - Look at camera, not screen (simulates eye contact)
    - Sit up straight, maintain energy
    - Hand gestures visible if possible
  - Contingency planning
    - Have interviewer's phone number ready
    - Test platform ahead of time (Zoom, Teams, etc.)
    - Practice screen sharing if presenting
