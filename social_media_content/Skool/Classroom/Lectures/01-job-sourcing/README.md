# Section 1: Job Sourcing Lectures (5 Methods)

*Source: The Launch Method V3 - Section 1*

## Videos to Record

### 1.1 The Job Sourcing Strategy™ ⭐ PRIORITY
- **Length:** 8-10 min
- **Status:** Not recorded
- **Key Points:**
  - Don't apply to 100s of random jobs hoping something sticks
  - Focus on quality applications to matching positions
  - Use GitHub repos for new grad opportunities (SimplifyJobs, PittCSC lists)
  - Monitor job boards specifically designed for new grads
- **Script:** See `00-launch-method-content-outline.md`

### 1.2 The Job Board Hierarchy™ ⭐ PRIORITY
- **Length:** 5-7 min
- **Status:** Not recorded
- **Key Points:**
  - LinkedIn (set up job alerts, most postings)
  - Indeed, Handshake (for students)
  - Company career pages directly
  - GitHub new grad lists (SimplifyJobs, PittCSC)
  - Glassdoor, ZipRecruiter, other aggregators

### 1.3 The Application Timing Strategy™
- **Length:** 5-7 min
- **Status:** Not recorded
- **Key Points:**
  - Apply within first 24-48 hours of posting
  - Set up daily job alerts for immediate notifications
  - Peak hiring seasons: January-March, September-October
  - Avoid holiday application dead zones

### 1.4 The Application Pipeline™
- **Length:** 5-7 min
- **Status:** Not recorded
- **Key Points:**
  - Track all applications in spreadsheet
  - Monitor follow-up timing
  - Categorize by stage: Applied → Phone Screen → Interview → Offer
  - Weekly pipeline review and maintenance

### 1.5 The New Grad Friendly Company List™
- **Length:** 5-7 min
- **Status:** Not recorded
- **Key Points:**
  - Companies known for strong internship-to-full-time pipelines
  - Organizations with structured new grad programs
  - Using SimplifyJobs and PittCSC GitHub lists
  - Researching company Glassdoor reviews for new grad experiences
