## üî§ FONTS

### Primary Font

**Plus Jakarta Sans** (Google Font)

1. In Canva: Go to **Brand Kit** > **Fonts** > **Upload a font** or search "Plus Jakarta Sans"
2. If not available in Canva, use **Poppins** or **DM Sans** as alternatives

### Font Weights to Add

| Weight | Use For |
|--------|---------|
| **Extra Bold (800)** | Main headlines, hero text |
| **Bold (700)** | Section headings, buttons |
| **Semi-Bold (600)** | Subheadings, emphasis |
| **Medium (500)** | Navigation, labels |
| **Regular (400)** | Body text |
| **Light (300)** | Large display text (optional) |

## üìê DESIGN ELEMENTS

### Corner Radius
- **Cards**: 12-16px (rounded-xl)
- **Buttons**: 9999px (fully rounded/pill shape)
- **Badges**: 9999px (fully rounded)
- **Icons**: 12px (rounded-lg)

### Standard Gradients

**Primary Gradient** (for backgrounds):
```
From: #064e3b (Brand Primary)
To: #14b8a6 (Brand Secondary)
Direction: Top to Bottom or Left to Right
```

**Accent Gradient** (for highlights):
```
From: #14b8a6 (Brand Secondary)  
To: #a3e635 (Brand Accent)
Direction: Left to Right
```

### Shadow/Glow Effects
- **Soft shadow**: Subtle drop shadow for cards
- **Glow effect**: Teal glow (`#14b8a6` at 25% opacity) for CTAs

### Logo Variations to Create in Canva

1. **Primary Logo** (for dark backgrounds)
   - White "L" icon + White text with lime "to" accent
   - Background: Transparent or Brand Primary gradient

2. **Light Logo** (for light backgrounds)
   - Emerald "L" icon on white square
   - Brand Primary text (`#064e3b`) with lime "to" accent

3. **Icon Only** (for profile pictures)

   - White "L" on emerald gradient background
   - OR Emerald "L" on white background
