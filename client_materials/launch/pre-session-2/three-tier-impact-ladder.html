The Problem

The Solution: Push your Result up the ladder. Each level signals something more impressive to the interviewer.

Example Progressions

EXAMPLE 1: Manufacturing Defect

1

"I fixed the manufacturing defect."

2

"I fixed the defect and created a checklist so it wouldn't happen again."

3

"I fixed the defect, created a checklist, presented it to the quality team, and it became the new standard operating procedure used across all shifts."

EXAMPLE 2: Bug Fix

1

"I debugged and fixed the production issue."

2

"I fixed the issue and added automated tests to catch it in CI/CD going forward."

3

"I fixed the issue, added tests, and led a blameless postmortem that resulted in new team practices for code review."

EXAMPLE 3: Process Improvement

1

"I built a dashboard to track our metrics."

2

"I built the dashboard and automated the data refresh so it's always up-to-date."

3

"I built the dashboard, automated refreshes, trained 3 other teams on using it, and it's now the company's standard for reporting."
