Minutes 75-82 | Slides 7-8

üé§ Mental Resilience Script

"Before we wrap up, I want to talk about the mental game. The job search is a marathon, not a sprint. You're going to face rejection‚Äîeveryone does. The difference is how you process it. The Rejection Reframe System is about treating the job search like a debugging exercise. Track everything. When you get rejected, look for patterns. Are you getting callbacks but not advancing? That tells you where to focus. You're not failing‚Äîyou're gathering data. And for Burnout Prevention: set a daily goal‚Äîmaybe 5 applications‚Äîthen STOP. Guilt-free rest is part of the system."

Rejection Reframe Key Points

  • ‚Ä¢ Track rejections as data, not failure
  • ‚Ä¢ Identify bottlenecks (resume ‚Üí phone ‚Üí interview)
  • ‚Ä¢ Decouple self-worth from outcomes
  • ‚Ä¢ "You're debugging, not failing"

"I'm getting interviews but no offers."
